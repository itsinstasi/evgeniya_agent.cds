Сохраняйте публикацию, кому важна

22 March 2019
0 0

Сохраняйте публикацию, кому важна данная информация 👆👆 ДЕКЛАРАЦИЯ О ДОХОДАХ 2018–2019 ГОДОВ В ИСПАНИИ: НОВОВВЕДЕНИЯ
В Испании 2 апреля стартует кампания по представлению декларации по подоходному налогу. Декларация может быть представлена онлайн или по телефону. Согласно календарю Налоговой службы, у налогоплательщиков есть время до 1 июля для того, чтобы отчитаться перед министерством финансов о своих доходах за 2018 год.
Одно из главных нововведений налоговой кампании этого года – освобождение от налогообложения выплат по уходу за ребенком для матерей и отцов. Данная мера вступила в силу после соответствующего решения, вынесенного Верховным судом в октябре 2018 года. Согласно данному постановлению, Министерство финансов Испании обязано вернуть подоходный налог, уплаченный с данных выплат, начиная с 2014 года.

Еще одно новшество этого года – увеличение вычетов для многодетных семей, где количество детей выше минимально необходимого для присвоения семье статуса многодетной. Вычет увеличивается до €600 в год за каждого ребенка.
Также добавляется вычет для тех, чья семья находится в другой стране ЕС.
